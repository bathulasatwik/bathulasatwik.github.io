Selected Projects

Gating Enables Curvature: A Geometric Expressivity Gap in Attention
A theory-driven project on geometric expressivity in attention architectures, studying how ungated attention yields structurally flat representation families under affine mixture structure, while multiplicative gating enables richer curved manifolds.
Decodability Is Not Sufficiency
A mechanistic interpretability study showing that highly decodable and ablation-sensitive features are not necessarily sufficient for correct model behavior. The project uses controlled counterfactual evaluation to distinguish sensitivity from sufficiency in representation-level analysis.
GitHub
Heat Kernel SVM on Manifolds
A geometry-aware classification project using support vector machines with heat kernels defined on manifolds. The implementation studies both a flat torus with an analytic Fourier-based heat kernel and a general surface setting using a mesh Laplacian and spectral heat kernel construction.
